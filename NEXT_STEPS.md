### Этап 1: Установка зависимостей

```bash
npm install
```

Установит:
- `axios` - для скачивания изображений
- `sharp` - для оптимизации изображений

---

### Этап 2: Скачивание изображений (ТЕСТ)

Сначала тестируем на Москве:

```bash
npm run download-images
```

**Что произойдет:**
- Скачаются все изображения Москвы (~848 шт)
- Сохранятся в `assets/images/moscow/{id}/{index}.jpg`
- Создается файл `data-generated/moscow_with_local_images.json`
- Объем: ~2-3 GB

**Время выполнения:** 10-20 минут
