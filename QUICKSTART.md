# 🚀 Быстрый старт

## ✅ Установка и запуск

### ⚠️ ВАЖНО: Требуется HTTP сервер!

Браузеры блокируют загрузку JSON файлов при открытии через `file://` из-за CORS policy.

**Необходимо запустить локальный HTTP сервер!**

### Способ 1: Быстрый запуск (Windows)

Просто запустите файл:
```bash
start-server.bat
```

Затем откройте в браузере: **http://localhost:8000**

### Способ 2: Через Python

```bash
python -m http.server 8000
```

Затем перейдите на **http://localhost:8000**

### Способ 3: Через Node.js

```bash
npx http-server -p 8000 -c-1
```

Затем перейдите на **http://localhost:8000**

### Проверка данных

Для проверки корректности категорий откройте:
**http://localhost:8000/check-data.html**

Это покажет детальную статистику по распределению заведений по категориям.

## 📊 Что работает

✅ **Просмотр городов** с реальным количеством заведений  
✅ **Просмотр категорий** (12 основных категорий)  
✅ **Фильтрация по городам**  
✅ **Фильтрация по категориям** (из всех городов)

## 🎯 Основные возможности

### Просмотр по городам
1. Кликните на **"Города"** в меню
2. Выберите нужный город
3. Увидите список всех заведений в этом городе

### Просмотр по категориям
1. Кликните на **"Категории"** в меню
2. Выберите категорию (например, "Рестораны")
3. Увидите все заведения этой категории из всех городов

### Фильтрация внутри города
1. Выберите город
2. Используйте поиск для фильтрации по названию
3. Используйте фильтр по категориям для уточнения

## 📁 Структура данных

```
catalog/
├── data/                    # ОРИГИНАЛЫ (не редактировать!)
│   ├── 1-cities.json       # Справочник городов
│   ├── 2-categories.json   # Справочник категорий
│   └── ...                 # Данные по городам
│
├── data-generated/          # РАБОЧИЕ файлы (генерируются)
│   ├── cities.json         # Города с count заведений
│   ├── categories.json     # Категории
│   └── ...                 # Города с category_id
│
├── script.js               # Логика приложения
├── styles.css              # Стили
└── index.html              # Главная страница
```

## 🔄 Обновление данных

Если нужно перегенерировать рабочие файлы:

```bash
node generate-working-data.js
```

Это автоматически:
- Проанализирует все заведения
- Присвоит категории на основе названия и особенностей
- Посчитает реальное количество заведений в каждом городе
- Сохранит результаты в `data-generated/`

## 📝 Заметки

- Оригинальные данные в папке `data/` **нельзя редактировать**
- Все изменения сохраняются в `data-generated/`
- При следующей генерации файлы в `data-generated/` будут перезаписаны
- 23.5% заведений не имеют категории (требуют ручной проверки при необходимости)
