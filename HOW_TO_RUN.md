# 🚀 КАК ЗАПУСТИТЬ ПРОЕКТ

## ⚠️ ВАЖНО: Проблема с категориями решена!

Если вы видите **"Категория не указана"**, это потому что:
- Браузеры блокируют загрузку JSON файлов при открытии через `file://`
- Это CORS policy - защита браузеров

## ✅ РЕШЕНИЕ: Запустите HTTP сервер

### Вариант 1: Быстрый запуск (рекомендуется)

Просто дважды кликните на файл:
```
start-server.bat
```

Или в терминале:
```bash
.\start-server.bat
```

Затем откройте: **http://localhost:8000**

---

### Вариант 2: Через Python

```bash
python -m http.server 8000
```

Откройте: **http://localhost:8000**

---

### Вариант 3: Через Node.js

```bash
npx http-server -p 8000 -c-1
```

Откройте: **http://localhost:8000**
