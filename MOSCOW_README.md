# Каталог заведений Москвы

JSON-файл с данными о заведениях Москвы для использования в веб-приложениях.

## Данные

Файл `data/moscow_companies_real.json` содержит информацию о 50 заведениях Москвы:
- Рестораны и кафе
- Магазины
- Сервисные центры
- Медицинские учреждения
- И другие категории

## Структура данных

Каждое заведение содержит:
- `id` - уникальный идентификатор
- `name` - название
- `address` - адрес
- `phone` - телефон
- `website` - веб-сайт
- `description` - описание
- `rating` - рейтинг
- `review_count` - количество отзывов
- `images` - изображения
- `features` - особенности
- `working_hours` - часы работы
- `social_links` - социальные сети
- `menu` - информация о меню/услугах
- `city_id` - идентификатор города

## Использование

```javascript
fetch('data/moscow_companies_real.json')
  .then(response => response.json())
  .then(companies => {
    console.log('Заведения Москвы:', companies);
  });
```

## Источник

Данные получены из Yell.ru парсера проекта YBase-TSP.
